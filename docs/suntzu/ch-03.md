# 謀攻

……破伍……19

……故上兵伐謀，其次伐交，其次伐兵，其下攻=城=攻城。攻城之法，脩修櫓、◇◇，三月而止；◇距◆，有又三月然後已。將不勝心之忿，而蟻傅之，殺士三分之一，而城不拔者，攻之𢦏災也。故善用兵者，詘屈人之兵而非戰也，拔人之城而非攻也，破人之國而非久也；必以全爭於天下，故兵不鈍而利可全，此謀攻之法也。

故用兵之法，什則圍之，伍則攻之，倍則分之，敵則能戰之，

……所以患軍…………知三軍…………◆既疑，諸侯之難至矣！是胃謂亂軍引勝。故知勝有五：知可而→以戰與不可而→以戰，勝；知眾……	……以虞侍待不虞，勝；將能而君弗御，勝。

凡此五者，勝之道也。故兵知彼知己，百戰不殆，不知彼而知己，一勝一負；不知彼不知己，每戰必殆。
