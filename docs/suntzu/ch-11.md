# 九地

孫子曰：地形者，兵之助。故用兵：有散地，有輕地，有爭地，有交地，有衢地，有重地，有覆地，有圍地，有死地。諸侯戰其地者，為散。入人之地而不深者，為輕。我得則利，彼得亦利者，為爭。我可以往，彼可以來者，為交。諸侯之地三屬，先至而得天下之眾者，為衢。入人之地深，背城邑多者，為重。行山林、險阻、沮澤，凡難行之道者，為覆。所由入者隘，所從歸者迂，彼寡可以擊吾眾者，為圍。疾則存，不疾則亡者，為死。是故，散地則無戰，輕地則無止，爭地則無攻，交地則無絕，衢地則合交，重地則掠，覆地則行，圍地則謀，死地則戰。

用兵有散地，有輕地，有爭地，有交地，有衢地，有重地，有泛地，有圍地，有死地。諸侯戰其地，為散。入人之地而不深者，為輕。我得則利，彼得亦利者，為爭。先至而得天下之眾者，為衢。入人之地深，倍背城邑多者，為重。行山林、沮澤，凡難行之道者，為泛。所由入者隘，所從歸者迂，彼寡可以擊吾眾者，為圍。疾則存，不疾則亡者，為死。是故，散地則毋戰，輕地則毋止，爭地則毋攻，交地則毋絕，衢地則合交，重地則掠，泛則行，圍地則謀，死地則戰。

所胃謂古善戰者，能使適敵人前後不相及也，◇◇◇◇◇……敢問：適敵眾以正，將來，侍待之若何？曰：奪其所愛則聽矣。兵之請情主數速也，乘人之不給也，由不虞之道，攻其所不戒也。

凡為客之道，深入則專，主人不克；掠於饒野，三軍足食。謹養而勿勞，并氣積力，運兵計謀，為不可賊測。投之毋無所往，死且不北，死焉不得？士民盡力。兵士甚陷則不懼，無所往則固；深入則拘，無所往則鬭。是故，不調而戒，不求而得，不約而親，不令而行；禁祥去疑，至死無所之。吾士無餘財，非惡貨也；無餘死，非惡壽也。令發之日，士坐者涕◇◇，卧者涕交頤。投之無所往者，諸、歲之勇也。

故善用軍者，辟譬如衛然。衛然者，恆山之蛇也；擊亓其首則尾至，擊亓其尾則首至，擊亓其中身則首尾俱至。敢問：賊可使若衛然虖乎？曰：可。越人與吳人相惡也，當亓其同周舟而濟也，相救若左右手。是故，方馬埋輪，不足恃也；齊勇若一，正政之道也；剛柔皆得，地之理也。故善用軍者，攜手若使一人，不得已也。將軍之事，靜以幽，正以治。能愚士之耳目，使無之→知；易亓其事，革亓其謀，使民無識；易亓其居，于迂亓其途，使民不慮。……深入諸侯之地，發亓其幾機。若敺驅羣羊，……	……九地之變，詘屈信伸之利，人請情之理，不可不察也。

凡為客，深則槫專，淺則散。◇國越竟境而師者，絕地也；四勶徹者，矍衢地也；入深者，重地也；入淺者，輕地也；倍背固前隘者，圍地也；倍背固前適敵者，死地也；毋無所往者，窮地也。◇◇◇◇散地，吾將壹亓其志；輕地，吾將使之僂；爭地，吾將使不留；交地也，吾將固其𥿍→結；衢地也，吾將謹亓其恃；重地也，吾將趣其後；泛地也，吾將進其途；圍地也，吾將塞其闕；死地也，吾將示之以不活。故諸侯之請情，遝則禦，不得已則鬭，過則從。

是故，不知諸侯之謀者，不能豫交；不知山林、險阻、沮澤之刑形者，不能行軍；不用鄉向道導者，不能得地利。四五者，一不智知，非王霸之兵也。彼王霸之兵，伐大國，則亓其眾不得聚；威加於適敵，則亓其交不得合。是故，不事天下之交，不養天下之權，信伸己之私，威加於適敵，故其國可拔也，城可隋隳也。无法之賞，無正政之令，犯三軍之眾，若使一人。犯之以事，勿告以言；犯之以害，勿告以利。芋之亡地然而后存，陷之死地131a而后生。大眾陷於害，然後能為敗為◇◇◇◇◇◇◇◇◇◇◇◇◇◇，千里殺將，此胃謂巧事。

是故，正征與舉之日，毋通其使；厲於郎廊上，以誅亓其事；適敵人開闠，必亟入之；先亓其所愛，微與之期，◇◇◇◇，以決戰事。是故，始如處女，適敵人開戶；後如脫兔，適敵不及距拒。
