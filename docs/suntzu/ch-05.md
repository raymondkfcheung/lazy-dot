# 勢

孫子曰：治眾如治寡，分數是；鬭眾如鬬寡，刑形名是也；三軍之眾，可使畢受適敵而无敗，◇正是也；兵之所加，如以段碫投卵，實虛是也。

凡戰，以正合，以奇勝；善出奇者，無窮如天地，无謁竭如河海。冬終而復始，日月是；死而復生，四時是。聲不過五，五聲之變，不可勝聽也；色不過五，五色之變，不可勝觀也；味不過五，五味之變，不可勝嘗也；戰埶勢不過奇正，奇正之變，不可勝窮也。奇正環還相生，如環之毋無端，孰能窮之？

水之疾，至於漂石者，埶勢也；執鷙鳥之擊，至於毀折者，節也。是以善戰者，其埶勢險，其節短；埶勢如彍弩，節如發幾機。

紛紛紜紜，鬬亂而不可亂；渾渾沌沌，形員圓而不可敗。亂生於治，脅怯生於恿勇，弱生於強。治亂，數也；恿勇脅怯，埶勢也；強弱，形也。善動敵者，形之，適敵必從之；予之，敵必取之。以此動之，以卒侍待之。

故善戰者，求之於勢，弗責於人，故能擇人而任埶勢。任勢者戰民也，如轉木石。木石之生性，安則靜，危則動；方則止，員圓則行。
