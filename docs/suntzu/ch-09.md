# 行軍

孫子曰：凡處軍、相敵：絕山依谷，視生處高，戰降毋登，此處山之軍也。平陸處易，而右倍背高，前死後生，此處陸之軍也。絕水，……此處水上之軍也。絕斥澤，唯亟去無留；為交軍沂→斥澤之中，依水草而背眾樹，此處斥澤之軍也。凡四軍之利，黃帝之所以勝四帝也。

凡軍好高而惡下，貴陽而賤陰；養生處實，是謂必勝，軍无百疾。陵丘、隄防，處亓其陽而右倍背之，此兵之利、地之助也。上雨水，水流至，止涉，侍待亓其定也。

絕澗，遇天井、天窖、天離、天◆、天郄，必亟去之，勿近也。吾遠之，敵近之；吾迎之，敵倍背之。軍旁有險阻、潢井、葭𥯤葦、小林、翳澮薈，可伏匿者，謹復索之，姦之所處也。

敵近而靜者，恃亓其險也；敵遠而挑人，欲人之進者，亓其所居者易利也。…………塵高而銳者，車來也；卑而廣者，徒來也；散而條達者，樵採者也；少而往來者，營軍者也。辤辭庳卑而備益者，進也；辤辭強而進敺驅者，退也；輕車先出居廁側者，陳也；無約而請和者，謀也。奔走陳兵者，期也；半進者，誘也。杖而立者，飢也；汲役先㱃飲者，渴也；鄉人見利而不進者，勞拳倦也；鳥襍集者，虛也。夜嘑呼者，恐也；軍獶擾者，將不重也；◇◇◇◇◇◇◇◇軍無懸◆甀者、不反返亓其舍者，窮寇也。◇◇閒閒，徐言人者，失亓其眾者也；數賞者，窘也；數罰者，困也；……兵怒而相迎，久而不合，又不相去也，必謹察此。

兵非多益，毋武進，……

卒未傅親而罰之，則不服，不服則難用也；卒已槫→傅親而罰不行，則不用。故合之以交→文，濟齊之以武，是胃謂102必取。令素行以教亓其民者，民服；素不行者，民不服。令素行者，與眾相得也。
