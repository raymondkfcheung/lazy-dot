# 作戰

孫子曰：凡用兵之法，馳車千駟，革車千乘，帶甲十萬。千里而饋糧，則外內之費，賓客之用，膠漆之財，車甲之奉，費日千金，然後十萬之師舉矣。

其用戰，勝久則鈍兵、挫銳，攻城則屈力，久暴師則國用不足；夫鈍兵、挫銳、屈力、殫貨，則諸侯乘其弊而起，雖智者，不能善其後矣。故兵聞拙速，未睹工久也；夫兵久而國利者，未有也。

故不盡於知用兵之害者，不能得用兵之利也。故善用兵者，役不再籍，糧不再載；取用於國，因糧於敵，故軍食可足也。國之貧於師者，遠者遠輸，遠者遠輸則百姓貧；近師者貴◇◇◇◇則◇及丘役。屈力中原，內虛於家，百姓之費，十去其六；公家之用，破車罷疲馬，甲冑矢弩，戟楯蔽櫓，丘牛大車，十去其七。故智將務食於敵，食敵一鍾，當吾廿鍾；萁秆一石，當吾廿石。故殺敵者，怒也；取敵之利者，貨也。故車戰，得車十乘以上，賞其得者，而更其旌旗；車雜而乘之，卒共而養之，是謂勝敵而益強。

故兵貴勝，不貴久。

故知兵之將，民之司命，國之安危之主也。
