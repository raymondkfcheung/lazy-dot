# 實虛

先處戰地而待戰者失佚，後處戰地而趨戰者勞；故善戰者，致人而不致於人。能使適敵自至者，利之也；能使適敵不得至者，害之也。故適敵失佚能勞之、飽能飢之者，出於亓其所必趨也；◇行千里而不畏，行无人之地也。攻而必取，攻亓其所不守也；守而必固，守亓其所必攻也。善攻者，適敵不知所守；善守者，敵不知所攻。微乎！微乎！◇◇◇◇◇◇◇◇，故能為適敵司命。進不可迎者，衝亓其虛也；退不可止者，而不可及也。故我欲戰，雖高壘深溝，適敵不得不與我戰者，攻亓其所必救也；我不欲戰，畫地而守之，適敵不得與我戰者，膠謬亓其所之也。故善將者，刑形人而无刑形，則我槫專而適敵分；我槫專而為壹，適敵分而為十，是以十擊壹也。我寡而適敵眾，能以寡擊眾者，◇◇◇◇◇◇◇◇地不可知，則適敵之所=備=者=多=所備者多；所備者多，則所戰者寡矣。備前者後寡，備左者右寡，无不備者无不寡=寡。寡者，備人者也；眾者，使人備己者也。知戰之日，知戰之地，千里而戰；不知戰之日，不知戰之地，前不能救後=後，後不能救前，左不能救右，右不能救左，皇況遠者數十里，近者數里乎？以吾度之，越人之兵雖多，亦奚益於勝𢦏哉！故曰：勝，可擅也；適敵唯雖眾，可毋𧯞鬭也。故績蹟之而知動靜之理，刑形之而知死生之地，計之而知63abc得失之策，角之知有餘不足之處。刑形兵之極，至於无=刑=无形；无形，則深閒間弗能規窺也，知智者弗能謀也。因刑形而錯措勝，眾弗能知也；人皆知我所以勝之形，而莫知吾所以制刑形。所以勝者不復，而應刑形無窮。夫兵刑形象水：水行，辟避高而走下；兵勝，辟避實擊虛。故水因地而制行，兵因敵而制勝。兵无成埶勢，无恆刑形，66能與敵化，之胃謂神。五行无恆勝，四時无常立；日有短長，月有死生。．神要
